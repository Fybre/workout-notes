# Support

## Getting Help

If you need help with Workout Notes, here are some resources:

- **Documentation**: Check the [README](README.md) for setup and usage instructions
- **Issues**: Search existing [GitHub Issues](https://github.com/Fybre/workout-notes/issues) or [create a new issue](https://github.com/Fybre/workout-notes/issues/new) for bugs or feature requests
- **Feature Requests**: Use the [GitHub Issues](https://github.com/Fybre/workout-notes/issues) with the "enhancement" label

## Before Creating an Issue

1. **Check the README**: Make sure you've followed all setup instructions
2. **Search existing issues**: Your question or problem might already be documented
3. **Check for updates**: Ensure you're using the latest version

## When Creating an Issue

Please provide as much detail as possible:

- **Environment**: OS version, device type, app version
- **Steps to reproduce**: Clear, step-by-step instructions
- **Expected behavior**: What you expected to happen
- **Actual behavior**: What actually happened
- **Screenshots**: If applicable, include screenshots or screen recordings

## Contributing

If you'd like to contribute to the project, please:

1. Fork the repository
2. Create a feature branch
3. Submit a pull request with a clear description

## Contact

For general questions or feedback, please use the GitHub Issues or Discussions.

## Support Policy

This is an open-source project maintained by volunteers. While we strive to respond to issues promptly, please be patient as we balance this project with other commitments.

## Security Issues

If you discover a security vulnerability, please report it privately by creating a [GitHub Security Advisory](https://github.com/Fybre/workout-notes/security/advisories) rather than filing a public issue.
